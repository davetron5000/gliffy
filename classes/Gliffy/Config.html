Class Gliffy::Config

  1. lib/gliffy/config.rb
Parent: Object

Global Configuration for Gliffy. This is a singleton currently and can be accessed via the config class method

  • You mostly need to set api_key, secret_key and account_name
  • You may wish to set protocol if you have a premium account and wish to use https

Attributes

account_name [RW] The name of your account
api_key [RW] Your API Key
gliffy_rest_context [RW] The url relative to gliffy_web_root of where the API is accessed
gliffy_web_root [RW] The gliffy web root, which is pretty much www.gliffy.com unless you know a secret
log_device [RW] The log device (as passed to Logger) for where log messages should go
log_level [RW] A Logger level to control logging
protocol [RW] The protocol, either ‘http’ or ‘https’ (though feel free to try ‘gopher:’ :)
secret_key [RW] Your Secret Key
